CSS3 Transform

Transform属性实现了一些可用SVG实现的同样的功能。可作用于内联(inline)元素和块级(block)元素。

Transform同时支持2D和3D (需配合perspective属性) 变换。此处我们暂时只讨论2D变换的情况。

  • Origin
  • Rotate
  • Scale
  • Translate
  • Skew
  • Matrix

贴士

CSS可视化格式模型描述了每个元素在坐标系统中的定位。在该坐标空间中，元素的定位和大小可以想象成像素的集合，从原点正直开始，到元素的最右和最下端。

Transform能够改变这个坐标系统。使得元素能够平移、旋转、缩放。

Origin

设置或检索对象以某个原点进行转换。默认值 center center。可以通过translate-origin属性来改变。

如果提供两个，第一个用于横坐标，第二个用于纵坐标。

如果只提供一个，该值将用于横坐标；纵坐标将默认为50%。

对应的脚本特性为transformOrigin。

transform-origin: [<percentage>|<length>left|right] [<percentage>|<length>top|bottom]

通过设置不同的transform-origin值，观察对相同角度转向rotate(45deg)的影响。

Original
0% 100%
20% 50%
right top
left bottom
100% 100%
80% 80%
.deg {
    -webkit-transform-origin: [percentage|length] [percentage|length];
       -moz-transform-origin: [percentage|length] [percentage|length];
        -ms-transform-origin: [percentage|length] [percentage|length];
         -o-transform-origin: [percentage|length] [percentage|length];
            transform-origin: [percentage|length] [percentage|length];
}

Rotate

基于角度转动一个对象，可用于内联元素和块级元素，需先有transform-origin属性的定义。

rotate(<angle>)

Original
30°
-30°
90°
120°
-120°
180°
.deg {
    -webkit-transform: rotate(angle);
       -moz-transform: rotate(angle);
        -ms-transform: rotate(angle);
         -o-transform: rotate(angle);
            transform: rotate(angle);
}

Scale

让元素变大或缩小。使用正数和负数以及小数作为参数。

使用负数值不会缩小元素，而是翻转它(比如，文字被反转)然后相应的缩放它。

使用0到1之间的小数来收缩或者缩小一个元素。

第一个参数对应X轴，第二个参数对应Y轴。如果第二个参数未提供，则默认取第一个参数的值。

scale(<number>[,<number>]), scaleX(<number>), scaleY(<number>)

Original
scale(.8,.8)
scale(-1,-1)
scaleX(2)
scaleY(2)
scale(2,-2)
scale(-2,2)
.scale {
    -webkit-transform: scale(x,y);
       -moz-transform: scale(x,y);
        -ms-transform: scale(x,y);
         -o-transform: scale(x,y);
            transform: scale(x,y);
}

Translate

基于X和Y坐标重新定位元素。参数可以为负数。

translate(<length>[,<length>]), translateX(<length>), translateY(<length>)

Original
translate(40px,10px)
translate(-40px,10px)
translateX(50px)
translateY(50px)
translate(40px,-10px)
translate(10px,40px)
.translate {
    -webkit-transform: translate(x,y);
       -moz-transform: translate(x,y);
        -ms-transform: translate(x,y);
         -o-transform: translate(x,y);
            transform: translate(x,y);
}

Skew

将对象倾斜，有两个参数(可以是负数)，分别代表x和y轴的倾斜度数。

将一个对象围绕着x和y轴按照一定的角度倾斜。这和rotate的旋转不一样，rotate只是旋转，而不会让元素的形状改变。skew会让一个元素的形状改变。

skew(<angle>[,<angle>]), skewX(<angle>), skewY(<angle>)

Original
skew(20deg,10deg)
skew(-20deg,10deg)
skewX(45deg)
skewY(-45deg)
skew(10deg,20deg)
skew(10deg,-20deg)
.skew {
    -webkit-transform: skew(angle,angle);
       -moz-transform: skew(angle,angle);
        -ms-transform: skew(angle,angle);
         -o-transform: skew(angle,angle);
            transform: skew(angle,angle);
}

Matrix

矩阵变换，基于X和Y坐标重新定位元素。

通过matrix函数，可以非常灵活的实现上述的各种效果。它有6个参数(a,b,c,d,e,f)，它事实上是一个3*3矩阵，经过该矩阵将旧的参数转换成新值。

matrix(<number>,<number>,<number>,<number>,<number>,<number>)

下面的图片形象的解释了矩阵数学原理。

Original
matrix(1,-1,1,1,200,40)
.matrix {
    -webkit-transform: matrix(a,b,c,d,e,f);
       -moz-transform: matrix(a,b,c,d,e,f);
        -ms-transform: matrix(a,b,c,d,e,f);
         -o-transform: matrix(a,b,c,d,e,f);
            transform: matrix(a,b,c,d,e,f);
}
